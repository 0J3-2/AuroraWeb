# Installing AuroraWeb

## Chrome

1. Download the Extension [here](https://github.com/0J3-2/AuroraWeb/releases/latest/download/AuroraWeb.crx)
2. Go to `chrome://extensions` in your browser
3. Enable `Developer Mode`
4. Drag and drop the downloaded `AuroraWeb.crx` file into the page
5. Click `Add Extension`
6. Done!
